Class MouseEventTransmitter

createjsのcanvasに対するMouseEventを下層に透過させるクラス。 stage上にマウスターゲットが存在しない場合、下層のDomElementにMouseEventを透過する。

skipMouseMovePerFrame: number = 2
  • Defined in MouseEventTransmitter.ts:21
